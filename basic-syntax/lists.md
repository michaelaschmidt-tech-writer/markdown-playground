# Lists

## Ordered lists

Create ordered lists by typing numbers and periods: `1.`, followed by text.<br>
For an indented list item, enter two tabs and again numbers and periods: `1.`.

1. First item
2. Second item
3. Third item
    1. First indented item
    2. Second indented item
        1. Indented item third level 1
        2. Indented item third level 2
    4. Third indented item
4. Forth item

## Unordered lists

Create unordered lists by typing either `-`, `+`, or `*`, followed by text.<br>
For an indented list item, enter two tabs and again `-`, `+`, or `*`.

- Item A, typed as `- Item A`
- Item B
- Item C

+ Item A, typed as `+ Item A`
+ Item B
+ Item C

* Item A, typed as `* Item A`
* Item B
    * Sub-item B.1
    * Sub-item B.2
* Item C

Tip: If you need to start an unordered list with a number followed by a period, e.g. "1087.", write like this: `- 1087\.` to get an unnumbered list:
- 1087\.
