# Code

## Backticks for code

To display text as code, enter the text in backticks ` `` `.

Here is an example for code: `code`

For displaying code that contains backticks, use douple backticks ` ```` `

Here is an example for code that contains backticks: ``This code contains `code`.``

## Code blocks

For code blocks, use three backticks ` ``` ` and indent every line by at least one tab:

```html
<html>
  <body>
    <p>Hello, world!</p>
  </body>
</html>
```
