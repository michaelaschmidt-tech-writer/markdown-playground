# Tables

## Tables with Markdown

You can create tables with three or more hyphens `---` as table headers and pipes `|` for the columns.

|Header 1|Header 2|
|---|---|
|Cell content 1|More content|
|Second line with content|Content|
||Content next to empty cell|

The text in Markdown does not have to be aligned, so you can type the above table like this:

`|Header 1|Header 2|`<br>
`|---|---|`<br>
`|Cell content 1|More content|`<br>
`|Second line with content|Content|`<br>
`||Content next to empty cell|`




## Tables with html

<table>
<thead>
<tr>
<th>foo</th>
<th>bar</th>
</tr>
</thead>
<tbody>
<tr>
<td>baz</td>
<td>bim</td>
</tr>
</tbody>
</table>
